Ambrosia Tutorial

A tutorial for installing Ambrosia (Point of Sale), configure and use it.

What is Ambrosia?​

Ambrosia is a modern Point of Sale (PoS) system. It allows businesses to accept Bitcoin payments through Lightning Network while managing inventory, sales, and business operations in one unified system.

What You'll Learn​

By following this tutorial, you'll be able to:

  • Install and configure Ambrosia on your machine
  • Manage users and products
  • Process your first Lightning payment

Requirements​

Cost:​

  • If you're just using a laptop
    • ~$FREE

Hardware​

Laptop​

  • Works on Linux, Mac, or (even!) Windows
  • Feel free to use one of these just to learn, though keep in mind you will want something that you can leave powered on and connected to the internet 24/7 if you really want to use this in a shop.

Software Prerequisites​

Before starting, you should have basic familiarity with:

  • Using a terminal/command line
  • Basic file navigation
  • Installing software on your operating system

Don't worry if you're new to Lightning Network or Bitcoin - we'll guide you through everything!

Tutorial Structure​

This tutorial is organized into several sections:

  1. Install Ambrosia - Installation
  2. Set Up Ambrosia - Configuration
  3. Make a Product - Products Management
  4. Sell Your Product - Processing Sales
  5. Other Features - Wallet, Users Management, Orders, Settings

Ready to Begin?​

Let's dive in and get Ambrosia running on your system! The next section will walk you through the installation process step by step.
