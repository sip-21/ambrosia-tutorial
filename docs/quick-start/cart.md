# Processing Sales

## 1. Open the Cart
- Dashboard → **Cart**
- Left: Products  
- Right: Order Summary + Payment

## 2. Add Products

### Method A — Browse
Click **Add** on any product card.

### Method B — Search
Type in the search bar:
```
Jade
```

### Method C — Category Filter
Click category → **Add** product.


## 3. Adjust or Remove Items

**Increase qty:** `+`  
**Decrease qty:** `-`  
**Remove:** Red trash icon

Totals update automatically.

## 4. Review Order
Check:
- Items  
- Quantities  
- Total amount  

## 5. Select Payment Method
Choose from dropdown:
- Cash  
- Credit Card  
- Debit Card  
- **BTC (Lightning)**

## 6. Complete Payment

### Cash / Card
1. Select method  
2. Click **Pay**  
3. Success → Cart clears

### Bitcoin (Lightning)
1. Select **BTC**  
2. Click **Pay**  
3. Customer scans the QR invoice  
4. Once paid, click **Payment Received**  
5. Success → Cart clears

## 7. Start Next Sale
Cart resets to $0.00.  
Inventory updates automatically.
