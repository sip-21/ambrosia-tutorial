# Managing Products

## 1. Open the Products Page
- From the Dashboard → **Products**
- You’ll see the list of existing products (name, price, stock, actions).

## 2. Add a Product

### Step 1 — Click "Add Product"
Top-right button.

### Step 2 — Fill the fields

**Product Name**
```
Jade Hardware Wallet
```

**Description**
```
Bitcoin hardware wallet
```

**Category**  
Create a new category.

```
Hardware Wallets
```

**SKU**
```
HW-JADE-01
```

**Price (MXN)**
```
2500
```

**Stock**
```
20
```

**Image (optional)**  
Upload a PNG/JPG image (square recommended).

### Step 3 — Click “Add”
Product appears immediately in the list.

## 3. Edit a Product
1. In the product list → click **Edit (pencil)**  
2. Change any field  
3. Click **Save**

Common quick edits:
- Increase stock  
- Change price  
- Update description

## 4. Delete a Product
1. Click **Delete (trash)**  
2. Confirm  

**Tip**: Instead of deleting, you can set stock to `0` if you might sell it again.
